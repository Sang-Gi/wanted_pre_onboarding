# 원티드 프리온보딩 코스

2022 원티드 프리온보딩 FE코스

## 프로젝트 실행 전 설치 명령어

—custom-component 폴더에서—
- npm 설치 : npm install
- CRA 설치 : npm install create-react-app
- yarn 설치 : npm install yarn --global
- sass 설치 : yarn add node-sass
- modal 설치 : yarn add react-modal

## 프로그램 실행
- custom-component 폴더에서 : yarn start
